## 🛠️ Get Started

1. Flash the ESP32 with firmware from `/data-logger`
2. Simulate machine data using Arduino sketch from `/Data-Generatore`
3. Connect the ESP32 to UART (TX/RX, GND, VCC)
4. View logs in Firebase!
